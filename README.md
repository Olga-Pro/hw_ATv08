# MCP Сервер Калькулятора
Сервер для математических вычислений с поддержкой Model Context Protocol, предоставляющий возможности калькулятора с историей операций.

## Возможности
✅ Базовые арифметические операции (+, -, *, /)

✅ Расширенные функции (sqrt, sin, cos, tan, log, exp)

✅ Математические константы (pi, e)

✅ История вычислений с временными метками

✅ Расширенные операции (факториал, степень, логарифм)

✅ Безопасная оценка выражений

## Установка
Скачайте или склонируйте проект

Установите зависимости:

bash
pip install -r requirements.txt
## Использование
### Как MCP Сервер
Сервер может быть подключен к MCP клиентам, таким как Claude Desktop.

### Поддерживаемые инструменты
calculate - Вычисление математических выражений

2 + 3 * 4

sqrt(16) + sin(pi/2)

get_history - Просмотр истории вычислений

Опциональный параметр limit для ограничения количества записей

clear_history - Очистка всей истории

advanced_calculate - Специализированные операции

факториал, возведение в степень, логарифм

### Примеры использования

calculate: "2 * (3 + 4) / 2"
Результат: 7.0

calculate: "sqrt(25) + sin(pi/2)"
Результат: 6.0

advanced_calculate: факториал 5
Результат: 120

advanced_calculate: логарифм 100 по основанию 10
Результат: 2.0
## Безопасность
Используется ограниченная оценка выражений для безопасности

Выполняется валидация входных выражений

Исключено произвольное выполнение кода

## Поддерживаемые операции
### Базовые операции
- Сложение: +
- Вычитание: -
- Умножение: *
- Деление: /
- Скобки: ()

### Математические функции
- Квадратный корень: sqrt(x)
- Тригонометрические: sin(x), cos(x), tan(x)
- Логарифмы: log(x) (натуральный), log10(x) (десятичный)
- Экспонента: exp(x)
- Степень: pow(x, y)

### Константы
- Число π: pi
- Число e: e

### Расширенные операции
- Факториал: n!
- Возведение в степень: x^y
- Логарифм с произвольным основанием: logₐ(x)

## Запуск сервера
bash
python calculator_server.py
Сервер запустится в режиме stdio и будет готов к подключению MCP клиентов.

## История вычислений
Сервер автоматически сохраняет все выполненные вычисления с:

- Выражением
- Результатом
- Временной меткой операции

История сохраняется в памяти до перезапуска сервера.

Примечание: Для использования с Claude Desktop необходимо добавить конфигурацию сервера в настройки MCP клиента.
